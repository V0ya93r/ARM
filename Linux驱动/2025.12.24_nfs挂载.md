### 按照教程来就行，以下是要注意的点
1. 虚拟机开启nfs服务且建好了共享文件后。首先设置开发板静态ip(每次开机自动设置好)
```bash
# 1. 编辑rcS脚本
vi /etc/init.d/rcS

# 2. 在脚本末尾添加以下内容（配置eth0网卡静态IP）
ifconfig eth0 192.168.8.22 netmask 255.255.255.0 up  # 设置IP和子网掩码，启用网卡
route add default gw 192.168.8.1  # 设置默认网关（可选，若需访问外网）

# 3. 保存退出（vi编辑器：按Esc，输入:wq回车）

# 4. 赋予脚本可执行权限（确保开机能执行）
chmod +x /etc/init.d/rcS
```
2.  mount -t nfs -o nolock,nfsvers=3 192.168.8.88:/home/voyager/Linux/nfs nfs/
ip是虚拟机的，两个地址一个是虚拟机共享文件夹，一个是开发板挂载的文件夹
3. ==自启动挂载==
```bash
# 编辑开发板的rcS自启脚本 
vi /etc/init.d/rcS 
# 在末尾添加以下内容，开机自动挂载NFS 
mkdir -p /home/nfs 
mount -t nfs -o nolock,nfsvers=3 192.168.8.88:/home/voyager/Linux/nfs /home/nfs
```
