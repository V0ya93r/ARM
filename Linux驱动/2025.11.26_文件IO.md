- 文件IO与标准IO：标准IO是间接调用C库函数来调用系统调用函数；而文件IO是直接调用内核提供的系统调用函数
- open函数：
	1. 文件描述符fd，error=-1，标准输入(鼠标键盘)=0，标准输出(显示器)=1，标准出错=2
	2. int open( const char * pathname, int flags);  
	   int open( const char * pathname, int flags, mode_t mode);
	   只有flags=O_CREAT时，mode_t参数才起作用
- close函数：
	1. close(fd)之后，fd释放并让内核重用
