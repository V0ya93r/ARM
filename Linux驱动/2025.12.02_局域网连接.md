LinuxUbuntu:
	ip 192.168.1.99 netmask 255.255.255.0 gateway 192.168.1.1 DNS 192.168.1.1
imx6ull:
	ip 192.168.1.22 netmask 255.255.255.0 gateway 192.168.1.1 DNS 192.168.1.1
- 连接方式：网线直连
- 注意：LinuxUbuntu可以直接在设置里修改网口的静态ip等，imx6ull须在/etc/network/interfaces中修改
```
iface eth0 inet dhcp
```
修改为
```
iface eth0 inet static
	address 192.168.1.22
	netmask 255.255.255.0
	gateway 192.168.1.1
	broadcast 192.168.1.255
```
